# randomHTMLapps

A grab bag of retro-flavored arcade experiments written in plain HTML, CSS and JavaScript. Each page is a self-contained mini game that runs entirely in the browser with no build step required.

## Games

- **galaxians.html** – Vertical shooter tribute with swooping invader formations and synth bleeps.
- **lander.html** – Lunar landing challenge with delicate thruster control and gnarly gravity.
- **vaders.html** – Space Invaders inspired defense with escalating waves and barriers.
- **breakout.html** – Neon brick breaker featuring power-ups, particle effects and multi-level progression.
- **pacman.html** – Maze-chasing pellet collector with wrap tunnels, a prowling ghost, and power-orb reversals.

Open any of the files directly in a browser or serve the directory with a static file server to play.
